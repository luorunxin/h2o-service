# h2o管理系统
> 从零架构，结构清晰，逻辑清晰

## 主要技术
> node + koa2 + koa-router + koa-websocket + jsonwebtoken + mysql

## 说明
1级目录下，h2o.sql文件是项目的数据库数据+结构，请启动mysql数据，使用Navicat工具运行sql，才能保证项目的数据交互
